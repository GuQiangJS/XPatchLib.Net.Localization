# XPatchLib.Net 多语言项目

## 编译

使用 Visual Studio 2017 及以上版本打开 `XPatchLib.Localization.sln` 。

因为项目文件使用了 Visual Studio 2017 中提供的新的 `csproj` 格式，所以 *开发者* 只能使用 Visual Studio 2017 及以上版本。

也可以在安装了 [Visual Studio 2017 生成工具](https://www.visualstudio.com/zh-hans/downloads/) 的前提下，执行 `build.bat` 进行编译。
